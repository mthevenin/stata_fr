Chargement de la biblio

Simplement avec bibliography: nom_biblio.bib, ici bibliography: marion2.bib.

1.2 Choix du style

Avec un fichier csl

  • On peut récupérer des styles de bibio: https://github.com/citation-style-language/styles. J’ai récupéré le style d’Economie & Statistique pour le format html
  • On colle le programme dans un fichier .txt et on l’enregistre avec l’extension .csl
  • Dans le header on appelle le style avec csl: nom_style.csl, ici csl: ecostat.csl dans le header.
  • En format pdf, il n’y aura pas de renvoi vers la bibliographie

Avec natlib ou biblatex

Pour le format pdf, on peut utilsier un style de biblio avec un fichier .cls mais on ne peut pas renvoyer la référence citée dans le texte vers la référence. On peut alors utiliser les styles biblatex ou natlib: cite-method: biblatex/natlib dans le header.

## Choix du style

**Avec un fichier csl**  

*  On peut récupérer des styles de bibio: https://github.com/citation-style-language/styles. J'ai récupéré le style d'*Economie & Statistique* pour le format html
*  On colle le programme dans un fichier .txt et on l'enregistre avec l'extension .csl
*  Dans le header on appelle le style avec `csl: nom_style.csl`, ici **`csl: ecostat.csl`** dans le header.
*  En format pdf, il n'y aura pas de renvoi vers la bibliographie

**Avec natlib ou biblatex**  

Pour le format pdf, on peut utilsier un  style de biblio avec un fichier .cls mais on ne peut pas renvoyer la référence citée dans le texte vers la référence. On peut alors utiliser les styles *biblatex* ou *natlib*: **`cite-method: biblatex/natlib`** dans le header.
