08 Septembre 2022: collect


collect La version 17 de Stata a introduit une suite nommée collect permettant de paramétrer, enregistrer, et utiliser des styles pour les tableaux. Dans le fichier d’aide, l’idée de remplacer les commandes utilisateurs existantes, en particulier la très populaire outreg2, est assez clairement annoncée. Après quelques heures d’apprentissage, je suis agréablement surpris pour ne pas dire bluffé.
Une première courte présentation pour les régressions se trouve dans le support programmation
